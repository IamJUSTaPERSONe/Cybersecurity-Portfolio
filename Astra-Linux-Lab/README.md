# Лабораторная работа: Развертывание VPN-сервера OpenVPN на Astra Linux

## 📋 О проекте
В рамках данной лабораторной работы была выполнена настройка защищенного VPN-канала между сервером на базе Astra Linux Common Edition и клиентом под управлением Windows 11.

## 🎯 Цели
- Установка и настройка серверной ОС Astra Linux в среде виртуализации VirtualBox
- Развертывание OpenVPN-сервера с использованием штатных средств Astra Linux
- Настройка клиентского подключения с Windows
- Диагностика и устранение типовых ошибок

## 🛠 Стек технологий
| Компонент | Технология |
|-----------|------------|
| ОС сервера | Astra Linux Common Edition 2.12 (релиз "Орёл") |
| Гипервизор | VirtualBox 7.x |
| VPN-сервер | OpenVPN + `astra-openvpn-server` |
| Клиент | Windows 10/11, OpenVPN GUI |
| Криптография | AES-256-GCM |

## 🚀 Этапы выполнения

### 1. Подготовка виртуальной среды
- Создание виртуальной машины в VirtualBox
- Настройка сетевых адаптеров: **NAT** (для доступа в интернет) и **Сетевой мост** (для связи с хостом)
- Установка Astra Linux Common Edition

### 2. Установка и настройка OpenVPN-сервера
```bash
sudo apt update
sudo apt install fly-admin-openvpn-server astra-openvpn-server -y
sudo astra-openvpn-server start
```
### 3. Создание клиентского сертификата
Сертификат был создан через графическую утилиту fly-admin-openvpn-server, что позволило избежать ошибок ручной генерации через easy-rsa.

### 4. Настройка клиента Windows
- Установка OpenVPN GUI
- Переброс файлов с виртуальной машины на клиента с помощью общей папки
- Подключение к серверу
