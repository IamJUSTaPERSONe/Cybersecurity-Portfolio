## USER ANALITICS FLASK APP

### ⚠️ ВНИМАНИЕ! Это учебный проект.

### 🚨Критические ограничения:
- **Не используйте `host='0.0.0.0'` в настройках IP хоста** - это откроет доступ для любого сетевого интерфейса в интернете >> компьютер будет открыт для атак!
- **Не собирайте реальные данные пользователей без их согласия** - это может привести к проблемам с законом (Федеральный закон от 27.07.2006 №152-ФЗ «О персональных данных»)
- **Используйте защищенный сервер для запуска приложение извне**


## 🛡️Безопасный запуск
```bash
# Активируйте виртуальное окружение 
venv\Scripts\activate 
# Установите зависимости
pip install -r requirements.txt
# Запустите приложение
python app.py
```

Приложение будет доступно в браузере по адресу http://127.0.0.1:5000 исключительно на вашем устройстве

### 🌐Локальный запуск
Если вы хотите дать доступ к приложению другим устройствам в локальной сети, необходимо изменить host на IP адрес вашего ПК:
```bash
if __name__ == '__main__':
    print('Запуск приложения')
    app.run(host='ВашРеальныйIP', port=5000, debug=False)
```

### 🔧Функциональность
- **Сбор данных о пользователе** (User-Agent, время посещения, IP и тд)
- **Простой интерфейс для ввода дополнительной информации**
- **Хранение данных в памяти** (данные теряются при перезапуске)

## ⚠️ Предупреждения для разработчиков
Если вы хотите использовать приложение извне, то используйте ngrok вместо прямого доступа (0.0.0.0)
ngrok http 5000
