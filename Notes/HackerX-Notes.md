1. От кейлогера можно защититься хорошим антивирусом, виртуальной клавиатурой и двухфакторной аутентификацией
2. Beelogger - ПО на линукс системах, которе через cmd может создать файл ворд, эксель, PDF и тп с кейлогером, которого можно отправить человеку с использованием соц. инженерии
3. С помощью ии можно создавать надежные пароли, проверять код на уязвимости, распозновать фишинговые письма
4. ПО sregosuite может использовать методы стеганографии для шифровки сообщений в изображениях. Сообщения, зашифрованные sregosuite можно извлечь только с помощью него
5. Сайт browseeleaks показывает все уязвимости, утекшие из браузера. Для предотвращения - впн
6. В опере есть встроенный впн без постороннего ПО
7. wormGPT - обячный ии, но созданный для взлома. Нет этики
8. Footprinting - разведка и сбор инфы о цели
9. Мак-спуфинг играет большую роль во взломе. Подмена мак может перехватить личность другого сетевого устройства, что часто используется в MitM
10. WPA/WPA2 (wifi protected access) очень похожи, единственное отличие - алгоритм шифрования информации. Можно взломать эксплуатацией WPS или с использованием словаря.
11. WPS (wifi protected setup) нужен для того, чтобы пользователи могли подключаться к сети без ввода ключа
12. При атаки MitM хакер, чтобы читать данные пользователя может понизить HTTPS соединение до HTTP, тк его очень сложно взломать
