# 🔐 Собственный прокси-сервер 

## 🎯 Цель проекта
Развертывание защищенного прокси-сервера для выборочного обхода блокировок с сохранением приватности критического трафика.

**Зачем?:** коммерческие VPN нестабильны, собирают логи и не дают контроля над инфраструктурой.

---

## 🏗️ Архитектура

| Компонент | Что использовано |
|-----------|------------------|
| Хостинг |  Aeza |
|Удаленное администрирование|PuTTY|
| Прокси-сервер | Xray-core |
| Панель управления | 3x-ui |
| Клиент | v2rayTun (Android) |
| Протокол | VLESS + Reality |

**Схема трафика:**
- Выбранные приложения/домены: трафик идет через сервер
- Банки, госуслуги, поиск → напрямую (локальный IP)

---

## 🛡️ Что сделано

### Безопасность личного кабинета на сайте хостинга
- Смена дефолтного пароля
- 2FA для входа
- Привязка к телеграмм

### Безопасность сервера
- Проверка конфигурации на безопасность выставленных настроек
-  Смена пароля root на более сложный 
- Настройка фаервола (только необходимые порты)

### Безопасность панели
- Смена дефолтных путей к панели управления (3х-ui) и подпискам
- 2FA при входе
- Смена логина/пароля администратора на более сложные

---

## 📚 Стек технологий
- `Linux (Ubuntu)` — базовая ОС сервера
- `PuTTY` - для администрирования сервера по SHH
- `Xray-core` — прокси-сервер
- `3x-ui` — веб-панель управления
- `VLESS + Reality` — протокол с маскировкой
- `uTLS` — имитация браузера
- `v2rayTun` — клиент для Android

---

## 🔧 Краткий процесс настройки

1. **Аренда VPS** (Подходящий тариф с минимальными комфортными характеристиками для обеспечения комфортной скорости сети)
2. **Установка 3x-ui** (установка скрпитом через PuTTY)
3. **Настройка inbound** (VLESS+Reality, порт 443, uTLS chrome)
4. **Создание клиента** (генерация UUID, Public Key, Short ID)
5. **Настройка клиента** (v2rayNG: подписка, раздельное проксирование)
6. **Безопасность:** смена паролей, 2FA, скрытые пути

---

## 📈 Результаты

- ✅ Полная автономность от коммерческих VPN
- ✅ Выбранный трафик проходит через собственный безопасный сервер, остальной напрямую через провайдера
- ✅ Реализована защита от DPI (Deep Packet Inspection)
