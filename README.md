# Cybersecurity-Portfolio

Привет! Меня зовут Настя, и это мое портфолио в области кибербезопасности. Здесь я собираю практичесикий опыт, исследования и интересные знания.

## 📁 Структура портфолио

- **FVWA-pentesting** - мои открытия в процессе взлома уязвимого приложения FVWA (Fancy Vulnerable Web Application) и краткое объяснение совершаемых атак.
- **Reports** - подробные отчеты о практикуемых атаках на уязвимое приложение.
- **My-Scripts** - мои собственные инструменты для демонстрации.
- **Notes** - заметки по курсам, собственным открытиям, разборы интересных кейсов.
- **TryHackMe-Notes** - заметки по комнатам на TryHackMe.
- **Astra-Linux-Lab** - практическая работа по администрированию впн-сервера на базе astra linux и подключение клиента по зашированному соединению

## 🎯 Цель
- Накопление и структурирование полезной информации.
- Демонстрация практического опыта в сфере кибербезопасности.
- Отслеживание прогресса и углубления знаний предметной области.
